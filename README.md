## Docker Server 服务

###### 简介
    基于 docker-compose 编写yaml文件，生成并快速构建系统服务,满足分布式服务服务环境的搭建以及测试环境服务环境的搭建
    
##### 其他信息
    默认包含的语言和其他中间件服务
    1、基于PHP的打包
    2、基于Golang语言编排
    3、包含Redis、MySQL
